Repdigit

A repdigit is a positive number composed out of the same digit.
Your task is to write a function that takes an integer and returns whether it's a repdigit or not.
Note The number 0 should return True (even though it's not a positive number).

Input

The first line of input contains the number.

Output

The first line displays “true” or “false”.

Sample Input Sample Output
66 true
0 true
-11 false
