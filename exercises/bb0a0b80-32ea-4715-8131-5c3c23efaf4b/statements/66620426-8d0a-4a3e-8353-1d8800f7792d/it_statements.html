Repdigit

Un repdigit è un numero positivo composto dalla stessa cifra.
Il tuo compito è scrivere una funzione che accetta un numero intero e restituisce se si tratta di un repdigit o meno.
Nota: Il numero 0 dovrebbe restituire True (anche se non è un numero positivo).

Input

La prima riga di input contiene il numero.

Output

La prima riga mostra "true" o "false".

Sample Input Sample Output
66 true
0 true
-11 false
