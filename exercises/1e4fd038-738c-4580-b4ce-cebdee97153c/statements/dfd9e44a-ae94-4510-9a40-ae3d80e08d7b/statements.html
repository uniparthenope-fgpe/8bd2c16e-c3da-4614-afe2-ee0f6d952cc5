Book Cover

The cover price of a book is € 24.
95, but a bookstore gets 40% off.
Shipping costs are € 3 for the first copy and 75 cents for each additional copy.
How much do n copies cost? Write a function final_price (n) that RETURNS the price.

Input

The first line contains the n copies.

Output

The first line contains the final price.

Sample Input Sample Output
24 379.53
0 0
10 159.45
