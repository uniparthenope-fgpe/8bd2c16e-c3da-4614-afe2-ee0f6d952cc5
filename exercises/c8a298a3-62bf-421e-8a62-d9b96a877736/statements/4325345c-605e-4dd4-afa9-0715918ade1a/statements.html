Sum of array

Your task is to calculate the sum of all elements of an array A of size N, given as input.

Input

The first line of input contains an integer N (1 < N ≤ 10), the size of array A. The second line of input contains the values of array A.

Output

The first line displays the sum of all values of array A if N fulfills the constraints, “Wrong input” otherwise.
